What to cut from scope

Remove for now: RFP automation is its own product. Don't build it in the same challenge submission — it competes with the Discovery flow for attention and triples the scope.

Keep as "phase 2" slide: TechZone auto-provisioning, Monday.com sync, Salesforce integration, Slack/Teams channels. Mention them as next steps — don't build them unless the core flow is already polished.

The differentiator isn't summarization — it's that the agent identifies gaps, flags risks, and routes work before generating anything. That's what separates it from "AI takes meeting notes."
